# codyssey_A1 — 프롬프트 관리 프로그램

터미널 기반의 콘솔 프로그램으로, AI 프롬프트를 등록하고 관리할 수 있는 프로그램입니다.
메뉴 번호를 입력해 프롬프트 추가, 조회, 검색, 즐겨찾기 관리 등의 기능을 사용할 수 있습니다.

## 실행 방법

```bash
python3 A1-1/code/main.py
```

실행 후 화면에 나타나는 메뉴에서 원하는 번호를 입력하세요.

## 기능 목록

| 번호 | 기능 | 설명 |
|---|---|---|
| 1 | 프롬프트 추가 | 제목, 내용, 카테고리를 입력하여 새 프롬프트 등록 |
| 2 | 프롬프트 목록 보기 | 등록된 모든 프롬프트를 번호와 함께 출력 |
| 3 | 카테고리별 조회 | 선택한 카테고리에 해당하는 프롬프트만 조회 |
| 4 | 프롬프트 검색 | 키워드로 제목/내용을 검색 |
| 5 | 프롬프트 상세 보기 | 번호를 입력해 프롬프트 전체 내용 확인 |
| 6 | 즐겨찾기 관리 | 즐겨찾기 추가/해제 및 즐겨찾기 목록 조회 |
| 0 | 종료 | 프로그램 종료 |

## 등록된 프롬프트 카테고리

- 텍스트 생성
- 이미지 생성
- 영상 생성
- 페르소나
- 자동화
- 기타

프로그램 시작 시 이전 과제에서 작성한 프롬프트 3개가 기본으로 등록되어 있습니다.
